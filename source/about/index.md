#### Programming Style

- 代码不写注释，我会尽量让她 self-explanatory，所以如果你看到代码中出现了注释，那么这段代码一定是我写的很烂，自己也弄不明白，~~亦或者是我用 Copilot 生成的~~。(￫ܫ￩)
- 不写 platform-dependent 代码，所有代码都是跨平台的，即插即用的。
- 对于编译器的 preference：和平台以及项目相关，MacOS 上我会使用 Clang，Windows 上我会使用 MSVC，Linux 上我会使用 GCC，如果是HPC项目，我会使用 Intel Compiler。
- 关于IDE：我会使用 JetBrains 的 IDE，因为我觉得他们的 IDE 很好用，不同语言的IDE之间的操作方式简直一摸一样，所以我不用花费时间去适应不同的 IDE。当然，我也会使用 VSCode，因为他的插件生态牛波一，特别是 Remote Development 唯一真神。（JB 的 Gateway 做的实在太难用了）
